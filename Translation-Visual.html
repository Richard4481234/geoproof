Translation

A translation slides every point of a figure by the same vector — the same distance in the same direction:

P′ = P + v  (the same v for every point)

So each point and its image are joined by an arrow, and all those arrows are equal in length and parallel.

What is preserved

Translation is a rigid motion (isometry): lengths and angles are unchanged and the image is congruent. Orientation is preserved too — ABC and A′B′C′ run the same way. In fact the image isn't even turned: it stays in the same "facing" as the original.

No fixed points

Unless the vector is zero, every single point moves — a translation has no fixed point (unlike a rotation, whose centre stays put, or a reflection, whose mirror line stays put).

Combining slides

Doing one translation then another is the same as a single translation by the sum of the two vectors. Translations also arise from reflecting across two parallel mirror lines.
